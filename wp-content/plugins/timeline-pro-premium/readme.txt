== Description ==

Timeline Pro  is pure HTML & CSS timeline style grid for WordPress.

Display any post type using short-codes anywhere.

Plugin Details: http://kentothemes.com/items/social/kento-responsive-timeline-for-wordpress/



<strong>Plugin Features</strong><br />

* Use via shortcode.
* Display any post type.
* Custom color.
* Ajax load more post.
* Ajax load post comments.
* Ajax comments reply .
* Featured post marker.
* Social Share button.
* Click to zoom view for readers.
* Support all update browsers including IE  8
* Support any themes
